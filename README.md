# YS Survey

## How to run?

1. Clone package
2. Run `npm install`
3. Run `npm run dev`

## Building the embedded version...

1. run `npm run build`
2. Copy the filename of the `app.[hash].js` generated file in /dist
3. Open `embed/index.html`.
4. Replace the filename there with the newly created.
5. Run `npm run embed`
